# FlightAware Delay/Cancellation Archive

This repository contains archived records of values publicly displayed on FlightAware delay and cancellation statistics pages.

## Scope

The records in this repository cover values displayed on the site from **April 2, 2026 through April 9, 2026**.

The dataset is intended as a reference archive of the values shown on the relevant pages during that period, including the U.S. delay and cancellation figures displayed at the time each entry was recorded.

## Notes on the Records

Entries were recorded at **15-minute intervals**.

If a page showed an error at a scheduled recording time, another recording was made **1 minute later**.

As a result, some entries may appear slightly offset from the regular 15-minute schedule.

## Time Standard

Timestamps in this repository are recorded in **GMT+8 / UTC+8**.

This is important when comparing the records to markets or discussions that refer to **U.S. Eastern Time**.

For example:

- **12:00 PM ET on April 7, 2026**
- corresponds to **12:00 AM UTC+8 on April 8, 2026**

Any review of these records should account for that timezone conversion.
